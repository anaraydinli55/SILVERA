# SILVERA E-commerce Website

This repository contains the front-end code for SILVERA, a premium and responsive silver jewelry e-commerce website. The platform aims to provide a luxurious, trustworthy, modern, and timeless online shopping experience for customers who value high-quality silver jewelry, elegant design, craftsmanship, and secure online transactions.

The website is built with a dark luxury and editorial design language, using a sophisticated color palette and elegant typography to create a high-end feel. It supports multiple languages (Azerbaijani, Russian, English) and features a comprehensive set of e-commerce functionalities.

## Technologies Used

*   **Frontend Framework:** React (with Vite for fast development)
*   **Styling:** Tailwind CSS (for utility-first CSS and responsive design)
*   **Routing:** React Router DOM
*   **Internationalization (i18n):** `react-i18next`
*   **Animations:** Framer Motion
*   **Icons:** Lucide React
*   **State Management:** React Context API (for Cart and Wishlist)

## Features

*   **Luxurious & Responsive Design:** Dark luxury theme with editorial elements, fully responsive for desktop and mobile.
*   **Multi-language Support:** Azerbaijani (Default), Russian, and English with instant language switching and persistence.
*   **Cinematic Hero Section:** Engaging full-screen hero visuals.
*   **Product Catalog:** Featured Collections, Best Sellers, and New Arrivals sections.
*   **Detailed Product Pages:** With images, descriptions, details, and related products.
*   **Shopping Cart & Wishlist:** Full e-commerce functionality for managing selected items.
*   **Secure Checkout:** Streamlined checkout process (mocked for frontend only).
*   **Brand Story & Trust:** Dedicated sections to highlight craftsmanship, quality, and trust factors.
*   **Customer Testimonials:** Showcase customer feedback.
*   **Newsletter Subscription:** For updates and exclusive offers.
*   **Floating WhatsApp Button:** One-click direct contact for inquiries and orders with pre-filled messages.
*   **Contact Page:** Dedicated section with contact information and a form.
*   **Policy Pages:** Privacy Policy, Terms & Conditions, Shipping & Returns.
*   **Smooth Animations:** Elegant scroll-reveal effects and micro-interactions using Framer Motion.
*   **Production-Ready Architecture:** Clean folder structure, reusable components, and scalability for future backend integration.

## Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm or Yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/silvera-ecommerce.git
    cd silvera-ecommerce
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the Development Server

To run the project in development mode:

```bash
npm run dev
# or
yarn dev
